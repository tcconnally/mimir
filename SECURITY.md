# Security Policy

## Supported Versions

Security updates are applied to the latest release. Please run the most recent
version of Mimir.

| Version | Supported          |
| ------- | ------------------ |
| >=0.2.0 | :white_check_mark: |
| <0.2.0  | :x:                |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please report security issues via email to **security@perseus.observer**.

### What to Expect

- **Acknowledgment:** We will acknowledge receipt within 48 hours
- **Investigation:** We will investigate and coordinate a fix
- **Disclosure:** Standard 90-day disclosure window. A fix will be published and
  advisory details released after the vulnerability is resolved or at the end of
  the disclosure period
